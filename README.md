# flavour-fit-meal-platform
Flavour Fit is a multi-role meal service platform designed for chefs, customers, delivery staff, and admins.
The system focuses on improving the food ordering experience through customized meal selection, clear user flow, and efficient service management.

## My Role
- Designed the UI/UX using Figma
- Created user flow and system workflow
- Defined business model and platform structure

## Features
- Chef and customer interaction
- Customized meal ordering
- Multi-role system (Chef, Customer, Delivery, Admin)
- Order and service workflow

## Tools
- Figma (UI/UX Design)
- System Design & Workflow Planning
